## 🎯 开宗明义

ggclaw 是一款脱胎于 OpenClaw 的 AI 桌面助手，基于 Rust 技术构建，完美兼容 OpenClaw Skills 生态。历经一个月快速迭代开发（3 年技术积累），专为提升个人和团队工作效率而设计。它不仅仅是一个 AI 聊天工具，更是一个**重新定义工作方式**的智能平台。

### 为什么选择 ggclaw？

- **🚀 极致性能**：采用自主研发的高性能架构，体积缩减 60%，启动速度提升 300%，内存占用降低 40%
- **🔒 隐私优先**：所有数据本地加密存储，零云端依赖，你的数据完全掌控在自己手中
- **🌐 跨平台**：一套代码，完美适配 macOS、Windows、Linux 三大操作系统，体验一致
- **🧩 生态兼容**：完美兼容 OpenClaw Skills 生态，无缝迁移现有插件，同时支持开发新的 Skills
- **💬 多平台集成**：无缝连接钉钉、飞书、Telegram、Discord 等主流 IM 平台，一处管理所有消息

## 🚀 快速开始

### 环境要求

- **操作系统**：macOS 10.15+、Windows 10+、Linux（主流发行版）
- **开发工具**：Node.js 18+、Rust 工具链
- **包管理器**：npm、pnpm 或 yarn

### 安装运行

我们提供了最简单的安装方式，三步即可开始使用：

```bash
# 1. 获取项目代码
git clone https://github.com/your-org/ggclaw.git
cd ggclaw

# 2. 安装依赖（自动检测并安装所有必需组件）
npm install

# 3. 启动开发环境
npm run tauri:dev
```

首次运行会自动下载所有依赖，整个过程大约需要 3-5 分钟，之后启动仅需 1.5 秒。

### 构建生产版本

我们针对不同平台进行了深度优化，构建出的应用体积小、性能强：

```bash
# macOS 通用版本（支持 Intel 和 Apple Silicon）
npm run build:mac

# macOS Intel 版本
npm run build:mac:x64

# macOS Apple Silicon 版本（M1/M2/M3）
npm run build:mac:arm64

# Windows 版本
npm run build:win

# Linux 版本
npm run build:linux
```

构建完成后，你会在 `src-tauri/target/release/bundle/` 目录下找到安装包。
